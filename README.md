Проект машинного обучения для решения задачи бинарной классификации: предсказание выживаемости пассажиров на борту «Титаника».
В рамках проекта реализован полноценный ML-пайплайн: от загрузки и предобработки данных до скрининга множества моделей, автоматического подбора гиперпараметров (Optuna), обучения лучшей модели и генерации файла для отправки на Kaggle.


Источник данных: Kaggle — Titanic: Machine Learning from Disaster
https://www.kaggle.com/c/titanic?spm=a2ty_o01.29997173.0.0.167355fbK3QuAp

### Описание задачи и данных
Цель: Предсказать, выжил ли пассажир (Survived: 1 или 0), на основе его демографических и экономических характеристик.

Основные признаки:
Pclass: Класс билета (социально-экономический статус).
Sex, Age: Пол и возраст.
SibSp, Parch: Количество родственников на борту.
Fare: Стоимость билета.
Embarked: Порт посадки.
Подробный разведочный анализ данных (EDA) и логика создания новых признаков описаны в ноутбуке first_EDA.ipynb.

### Структура проекта

Проект построен по принципу разделения ответственности (Separation of Concerns). Каждый модуль выполняет одну конкретную задачу.

🚀 final_project_titanic/
 │
 ├─ 📄 main.py                 # Точка входа. Запускает весь пайплайн обучения.
 ├─ 📄 config.py               # Централизованная конфигурация (OmegaConf). Пути, параметры моделей, флаги.
 ├─ 📄 data.py                 # Загрузка train.csv и test.csv, разделение на X и y.
 ├─ 📄 features.py             # Конструирование признаков (Initial, Family_Size, Alone).
 ├─ 📄 preprocessing.py        # Обработка пропусков, кодирование категориальных признаков, масштабирование.
 ├─ 📄 models.py               # Фабрика моделей. Инициализация всех алгоритмов с параметрами из конфига.
 ├─ 📄 evaluate.py             # Логика кросс-валидации (StratifiedKFold) и оценка качества.
 ├─ 📄 tuning.py               # Модуль подбора гиперпараметров с помощью библиотеки Optuna.
 ├─ 📄 training.py             # Оркестратор: скрининг, тюнинг, финальное обучение, предсказание и сохранение.
 ├─ 📄 NeuralNet.py            # Кастомная нейросеть на PyTorch (MLP) с оберткой под sklearn и Early Stopping.
 ├─ 📓 first_EDA.ipynb         # Jupyter Notebook с разведочным анализом данных.
 ├─ 📂 datas/                  # Директория для исходных данных (train.csv, test.csv).
 └─ 📂 output/                 # Директория для результатов (создается автоматически).
     ├─ metrics/               # CV-результаты (csv) и сводка (json).
     ├─ models/                # Сериализованные модели (.joblib).
     └─ submissions/           # Файлы для отправки на Kaggle (.csv).
